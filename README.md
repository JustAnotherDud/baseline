# Baseline
Personal athlete data PWA

Mobile-first dark-theme PWA for nutrition tracking and body composition. No build step, no framework.

## Stack

- HTML / CSS / JS vanilla
- Supabase JS v2
- GitHub Pages

## Modules

- Diário nutricional
- Base de alimentos
- Templates de refeições
- Targets modulares (DCB)
- Estatísticas
- Forma — composição corporal (Garmin) + treino (Intervals.icu) + ginásio (Hevy)
